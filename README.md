# Note App CRUD

**Prerequisites**
- Python 3
- Python Package Installer (PIP)
- Virtual Environment (recommended)

**Running a Development Server**
1. Create a virtual environment (recommended) for the project.
    ```bash
    $ python3 -m venv venv
    $ source venv/bin/activate
    ```
2. Install the packages specified inside `requirements.txt`.
    ```bash
    (venv)$ pip3 install -r requirements.txt
    ```
3. Run the application
    ```bash
    (venv)$ python3 app.py
    ```
4. Access the [application](http://127.0.0.1:5000) via a web browser on localhost at port `5000`.
